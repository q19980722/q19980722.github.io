# Zifeng Wang Contact Page

A static GitHub Pages contact page for Zifeng Wang.

## Files

- `index.html`: The page markup and small copy-to-clipboard interaction.
- `styles.css`: The responsive visual design.
- `.nojekyll`: Keeps GitHub Pages from applying Jekyll processing.

## Deploy On GitHub Pages

1. Create or open the GitHub repository you want to use.
2. Commit these files to the branch you will publish, usually `main`.
3. In GitHub, go to `Settings` -> `Pages`.
4. Set the source to `Deploy from a branch`, choose your branch, and use `/ (root)` as the folder.

The page will be available at the GitHub Pages URL shown in that settings page.
